** Architecture (leaf → outward)
- =internal/domain= — entities + enums, no SQL/HTTP. (entities.go, enums.go)
- =internal/store= — libSQL queries + goose migrations (0001..0009).
- =internal/service= — business rules (sprint health, cadence, metrics, traces).
- =internal/block= — block registry + per-user enable/disable (registry.go, service.go).
- =internal/auth= — login, sessions, API keys, middleware.
- =internal/onboarding= — 3-step wizard (role → blocks → confirm).
- =internal/bridge= — WebSocket ↔ kiro-cli chat proxy.
- =internal/export= — server-side PDF (fpdf).
- =internal/web= — handlers, html/template pages, static CSS/JS, REST API v1.
